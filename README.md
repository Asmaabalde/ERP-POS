# 🧩 PlanifyX — ERP & POS Intelligent Platform  
Projet de fin d’année (L3 Informatique) — Université Paris Cité  
Développé en groupe de 4 étudiants (2025–2026)

PlanifyX est une plateforme complète combinant un **ERP (Enterprise Resource Planning)** et un **POS (Point of Sale)** moderne, conçue pour les petites et moyennes entreprises.  
Ce projet constitue le **projet de fin de Licence** réalisé en groupe de quatre étudiants dans le cadre du module L3K1 à l’Université Paris Cité.

L’objectif est de proposer une solution centralisée, modulaire et intuitive permettant de gérer l’ensemble des opérations d’une entreprise : clients, produits, stocks, fournisseurs, commandes, employés, factures, analytics et caisse.

---

## 🎯 Objectifs du projet

- Centraliser la gestion des données de l’entreprise dans un système unique.  
- Proposer un ERP modulaire, extensible et personnalisable.  
- Intégrer un POS complet, rapide et utilisable hors ligne.  
- Garantir la cohérence, la sécurité et la disponibilité des données.  
- Offrir une interface moderne, ergonomique et adaptée aux utilisateurs non techniques.  
- Fournir une architecture robuste permettant l’évolution future du système.

---

## 🧠 Fonctionnalités principales

### 🔹 ERP — Gestion interne
- **Clients** : ajout, modification, recherche, suppression, historique d’achats, fidélité.  
- **Produits & catalogue** : variantes, catégories, prix, TVA, code-barres, filtres.  
- **Stocks avancés** : mises à jour automatiques, alertes, réapprovisionnement, tendances.  
- **Fournisseurs** : coordonnées, délais, historique, commandes, paiements.  
- **Commandes & achats** : création, réception, scan, mise à jour du stock.  
- **Factures** : génération PDF, envoi par mail, conformité légale.  
- **Analytics** : tableaux de bord (ventes, clients, produits, fournisseurs, employés).  
- **Ressources humaines** : gestion des employés, rôles, salaires, fiches de paie.  
- **Administration** : création d’utilisateurs, attribution de rôles, gestion des accès.

### 🔹 POS — Point de vente
- Interface de caisse rapide et intuitive.  
- Paiement multiméthodes (carte, espèces, fidélité).  
- Suggestions intelligentes (produits complémentaires, offres personnalisées).  
- Gestion des remises, coupons, fidélité.  
- Mode hors-ligne avec synchronisation automatique.  
- Impression et envoi des tickets/factures.

## 🏗️ Architecture & technologies

### 🔸 Backend
- **Python — Django (MTV)**  
- API REST structurée  
- PostgreSQL (types Decimal pour précision financière)  
- Gestion des rôles, permissions, sécurité, logs  
- Hébergement prévu sur serveur universitaire ou local

### 🔸 Frontend
- **React + Vite**  
- Architecture basée sur les composants  
- Stores pour la gestion d’état global  
- Interfaces modernes et responsives

### 🔸 Outils & workflow
- Subversion (SVN) — gestion de versions  
- PyCharm & VS Code  
- Documentation complète (charges, recette, conception, tests, manuels)  
- Packaging final (Semaine 11)  
- Soutenance & livrables (Semaine 12)
